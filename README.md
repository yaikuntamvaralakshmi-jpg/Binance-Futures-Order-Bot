# Binance-Futures-Order-Bot
Binance Futures Order Bot is a Python CLI trading system for automating Market, Limit, Stop-Limit, OCO, TWAP, and Grid orders on Binance USDT-M Futures. It features input validation, structured logging, modular API adapters, unit tests, and CI workflows, enabling safe, reliable, and extensible automated trading.
